abstract void SetGameOver ( string  gameOverText)
pure virtual

Sets the game as game over and shows a custom text. For Campaigns this will start the campaign all over. For Versus maps this will indicate that the current survival wave is over, if all players are dead the current wave will restart, if someone's alive it will progress. For Challenges this will end the challenge, the Game.ChallengeCompleted property determines if the challenge is completed successfully or not.

Parameters
gameOverTextCustom game over text.
