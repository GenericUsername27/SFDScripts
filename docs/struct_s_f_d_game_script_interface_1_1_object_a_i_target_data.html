Member Data Documentation

◆ PriorityModifier

float PriorityModifier

The higher this value is, the more likely the bot will attack this object and ignore other objects based on distance.

◆ Range

float Range

Max range for bots to consider targeting this object. A negative or zero value will be considered infinite range. Note: Bots update in a round robin pattern and not every update. Setting this value too small and bots will ignore and walk past the object. Note: Bots have an internal range modifier to avoid easy cyclic behaviors.

◆ TargetMode

ObjectAITargetMode TargetMode

Target mode for the bot. If set to range only the bot will only care about this object if it has a ranged weapon (firearm). If set to melee only the bot will ignore to use ranged weapons against this target.

◆ Team

PlayerTeam Team

The team the object belongs to. The bot will ignore this object if it's in the same team.
